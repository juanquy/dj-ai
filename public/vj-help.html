DJAI VJ Visualizer Help

The DJAI VJ Visualizer allows you to add professional VJ-style visual effects to your AI mixes. These visuals react to the music in real-time, creating an immersive audio-visual experience.

Getting Started

The VJ Visualizer is automatically enabled when you generate a mix. You can control it using the floating panel in the bottom right corner of the screen.

Tip: If you don't see the visualizer, check if it's enabled by toggling the "Enable Visuals" switch in the VJ Controls panel.

Visualization Types

You can choose from several different visualization styles:

Visualization Description
Waveform Classic audio waveform display that reacts to music amplitude
Particles Dynamic particle system that pulses and flows with the beat
Spectrum Frequency spectrum analyzer with colorful bars
Tunnel 3D tunnel effect that pulses to the beat
Gradient Flowing color gradients that morph with the music

Performance Tips

The VJ Visualizer is designed to work on most modern devices, but complex visualizations can be resource-intensive. Here are some tips to improve performance:

  1. Use the "Auto" quality setting to automatically adjust based on your device's performance
  2. Disable "Glow Effect" on lower-end devices
  3. The "Particles" visualization is the most resource-intensive; use others if you experience lag
  4. Video backgrounds can be disabled to improve performance
  5. Reduce the browser window size if needed for smoother performance
Note: The visualizer uses WebGL for some effects when available. If your device doesn't support WebGL, the visualizer will automatically use Canvas-based rendering instead.

Keyboard Shortcuts

Key Action

Troubleshooting

If you encounter issues with the visualizer:

  • Make sure your browser is up to date
  • Try refreshing the page
  • Disable other browser extensions that might interfere
  • Use the "Low" quality setting on older devices
  • If the visualizer isn't appearing, check if hardware acceleration is enabled in your browser
